Job Description

We're looking for a talented Frontend Developer to join our growing engineering team. You'll be responsible for building responsive, accessible user interfaces that power Acme's core products and serve millions of users worldwide. This role offers the opportunity to work with cutting-edge technologies and collaborate closely with our design and product teams.

What makes this role exciting: You'll be working on products that directly impact user experience and have the freedom to influence technical decisions and architectural choices.

Key Responsibilities

  • Build and maintain responsive web applications using React and TypeScript
  • Collaborate with designers to implement pixel-perfect, accessible user interfaces
  • Optimize application performance and ensure cross-browser compatibility
  • Write clean, maintainable code with comprehensive testing
  • Participate in code reviews and contribute to engineering best practices
  • Work closely with backend engineers to integrate APIs and services
  • Stay up-to-date with frontend technologies and share knowledge with the team

Experience Requirements

  • 3+ years of professional frontend development experience
  • Strong portfolio demonstrating modern web application development
  • Experience with component-based architecture and state management
  • Familiarity with agile development processes
  • Understanding of web performance optimization techniques
  • Experience working in collaborative, cross-functional teams

Our Engineering Culture

At Acme, we believe great software comes from great teams. Our engineering culture emphasizes:

  • Code Quality: We value clean, well-tested code and thorough code reviews
  • Continuous Learning: Regular tech talks, hackathons, and knowledge sharing
  • Innovation: 20% time for exploring new technologies and side projects
  • Collaboration: Cross-functional teams working toward shared goals
  • Impact: Your work directly affects millions of users worldwide
Fun fact: Our frontend team has contributed to several open-source projects and regularly speaks at conferences. We'd love for you to be part of that community involvement!
